Create personal layer files for all .lyr files from a source folder. Designed for use with a password protected geodatabase. Afterwards using the personal layer files will skip authentication prompt.

For obvious reasons don't share the personal layer files.

Source path: folder containing layer files to be converted. Subfolders are processed as well.

SDE connection: location of database connection .SDE file with saved credentials.

Output path: where to save the personal layer files. Your personal documents folder is a good location. Layers will be grouped by servername of the sde file and retain their source folder hierarchy.

Group layers are skipped.
